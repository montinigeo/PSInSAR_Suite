about=PSInSAR Suite is an integrated QGIS plugin that brings together four tools for PS-InSAR (Persistent Scatterer Interferometric SAR) data analysis into a single dedicated toolbar.

 - PSInSAR Load: loads PS-InSAR point layers from GeoPackage, Shapefile or GDB files using a polygon index layer, or reactivates an index already present in the project.
 - PSInSAR EWUD: decomposes LOS velocities from ascending/descending Sentinel-1 (or other SAR) data into East-West and Up-Down components, with automatic resampling grid creation. Outputs include Na and Nd fields (number of PS points used per cell).
 - PSInSAR VIS: calculates the detectable movement percentage (pc_mov) based on SAR acquisition geometry and terrain morphology derived from a DEM.
 - PSInSAR TS: time series analysis tools including normality check, automatic kinematic analysis with mean series layer output, seasonal decomposition (STL), piecewise non-linear analysis, and geostatistical kriging.
